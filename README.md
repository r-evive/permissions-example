# 🔐 Permissions Example (Next.js)

This project is a **simple example of a permissions system** in a Next.js application.

> ⚠️ Client-side only — does NOT provide real security.

---

## 🌍 Demo

https://permissions-example.vercel.app

---

## 🎯 Purpose

- Show how permissions can control UI and navigation
- Demonstrate basic access logic structure

---

## 🚫 Limitations

- No authentication
- No server-side protection
- Session stored on client (can be bypassed)

---

## 🚀 Run locally

```bash
npm install
npm run dev
